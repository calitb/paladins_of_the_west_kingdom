# Paladins of the West Kingdom

## Setup

### Install dependencies

```shell
npm i
```

## Running the App

### Start Expo

```shell
npm start
```

## Run Linter

```shell
npm run lint
```

## Run Typescript check

```shell
npm run ts
```
